Projects:

DSS Engine:

-The DSS Engine is an ongoing project which attempts to create a fully modular 2D physics engine in Python using the Pygame graphics module.

-This large project was started in July 2022 alongside two of my peers, and one of the main goals of the project was to familiarize ourselves with low level graphics and rendering processes in game development

-The engine is extremely lightweight, boasting a size of only 388 kilobytes!

-Click here to go to the project's Github repository

Missile Command:

-Missile Command was a group semester project in our high School Video Game Programming class.

LT Space Game:

-LT Space Game (working title) was a large video game group project I started in my senior year of high school, the aim being to create a 2D RPG.

-I wanted a way for people interested in different fields (art,music,design,and programming) to come together and apply their knowledge in a tangible project, so I decided to start it as a school club

-Everyone was cooardinated into different teams based on their interests and over the span of the school year we all worked towards getting a finished project

-Due to time constraints, as we were all still full time high school students, parts of the project remain unfinished however by the end we had a working prototype which was the culmination of everyone's continued efforts
